Tableturf Bot Terms of Service

Last updated: 2024-08-12

Proper Usage

Tableturf Bot is a Discord Bot for various Tableturf Battle and TableturfQ utilities.
You may use any commands within a server as long as it doesn't break Discord's TOS.

Tableturf Bot Privacy Policy

Last updated: 2024-08-12

Proper Usage

Tableturf Bot collects basic user data from users that logs into TableturfQ with Discord.
The access token and refresh token will never be shared, and can be revoked by users at any time by manually logging out from TableturfQ's profile page.
Tableturf Bot does not collect data from Discord servers it is a member of, and Tableturf Bot does not collect data from the messages within those servers.
